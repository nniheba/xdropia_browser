# Instrucciones para Probar XDropia Browser

## Instalación

1. Descargue el archivo **XDropia-Browser-Windows.zip** desde [GitHub Releases](https://github.com/nniheba/xdropia_browser/releases/tag/v1.0.0)
2. Descomprima el archivo en la ubicación deseada
3. Ejecute **XDropia Browser.exe** para iniciar la aplicación

## Cuentas de Prueba

Puede utilizar cualquiera de las siguientes cuentas para probar diferentes niveles de acceso:

### Cuenta Free
- **Email**: usuario.free@xdropia.com
- **Contraseña**: XDropia2023Free
- **Acceso**: Herramientas básicas (Canva básico)

### Cuenta Premium
- **Email**: usuario.premium@xdropia.com
- **Contraseña**: XDropia2023Premium
- **Acceso**: Herramientas premium (Canva, ADSParo, ChatGPT)

### Cuenta Team
- **Email**: equipo@xdropia.com
- **Contraseña**: XDropia2023Team
- **Acceso**: Todas las herramientas con soporte para 5 usuarios simultáneos

### Cuenta Admin
- **Email**: admin@xdropia.com
- **Contraseña**: XDropia2023Admin
- **Acceso**: Todas las herramientas + Panel de Administración

## Funcionalidades por Tipo de Cuenta

### Usuario Free
- Acceso a herramientas básicas
- Una sesión activa a la vez
- Sin acceso a herramientas premium

### Usuario Premium
- Acceso a todas las herramientas disponibles
- Una sesión activa a la vez
- Acceso a todas las credenciales

### Usuario Team
- Acceso a todas las herramientas disponibles
- Hasta 5 sesiones activas simultáneamente
- Ideal para equipos de trabajo

### Administrador
- Acceso completo a todas las herramientas
- Panel de administración para gestionar credenciales
- Estadísticas de uso y usuarios
- Capacidad para añadir/modificar credenciales

## Notas Importantes

- Las credenciales de servicios están ocultas por seguridad
- No es posible inspeccionar el navegador para extraer información sensible
- Solo se permite cambiar contraseñas dentro de la aplicación
- No se permite modificar el correo electrónico
